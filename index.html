Как начать ей пользоватся?

вставте вот эту ссылку в ваш терминал wget https://raw.githubusercontent.com/lord-Vampirion-Tech/cct-gui-lib/main/GUI.lua GUI.lua и библиотека установится сама с репозиторя Git Hub https://github.com/lord-Vampirion-Tech/cct-gui-lib

в своей программе просто пропишите local gui = dofile("GUI.lua") и можете пользоватся всеми благами библиотеки. Все модули, предоставляемые библиотекой, отображаются в боковой панели.

GUI.Manager

Данный класс позваляет работать сразу с несколькими обьектами добавляя их группы

Метод Описание
:add(element, ...) Добавить элемент в одну или несколько групп.
:delAll(nameGroup) Удалить все элементы из указанной группы.
:delIf(nameGroup, predicate) Удалить элементы из группы, если они соответствуют условию.
:setColors(nameGroup, colors) Установить цвета фона и текста для всех элементов в группе.
:setProperty(nameGroup, prop, value) Изменить свойство у всех элементов группы.
:callAll(nameGroup, action, ...) Вызвать метод у всех элементов группы (например, onClick).

Добавить элемент в группу(ы).

GUI.Manager:add(element, ...)

Описание:

  • element - Это то, что вы хотите добавить в группу
  • ... - одна и более групп в которые вы хотите добавить element

Пример: 

-- создаем объект типа Label
local Label = {...}

-- а затем добавляем его в группу base
GUI.Manager:add(Label, "base")

доп. информациооное

Каждый из обьектов библиотеки можно прям при его создании добавитть в группы

-- покажем на примере Label
local Label = {свойства, ...} -- троиточие, есть список групп

Удалить всю группу

GUI.Manager:delAll(nameGroup)

Описание:

  • nameGroup - имя группы, что вы хотите удалить

Пример: 

GUI.Manager:delAll("base") -- удаляем всю группу base

Удалить элементы группы по определенным условиям, которые вы укажите.

GUI.Manager:delIf(nameGroup, predicate)

Описание:

  • nameGroup - имя группы, что вы хотите удалить
  • predicate - условие, по которому будут отбиратся объекты

Пример: 

-- пример удаления по типу
GUI.Manager:delIf("base", function(e)
    return e.type == "Button"
end)

Установить цвет для элементов группы

GUI.Manager:setColors(nameGroup, colors)

Описание:

  • nameGroup - имя группы, что вы хотите удалить
  • colors - цвета фона и текста

Пример: 

GUI.Manager:setColors("base", {
    back = colors.black,
    text = colors.blue,
})

примечание: colors принемает 2 параметра, как указанно в примере (можно указать только один из них)

Изменить своство у элементов группы по определенным условиям

GUI.Manager:setProperty(nameGroup, prop, value)

Описание:

  • nameGroup - имя группы, что вы хотите удалить
  • prop - какое свойсвтво у объектов вы хотите поменять
  • value - новое значение свойства
-- обьекты типа Button
local Button_1 = {...}
local Button_2 = {...}
local Button_3 = {...}

-- добавляем Button's в группу base
GUI.Manager:add(Button_1, "base") 
GUI.Manager:add(Button_2, "base")
GUI.Manager:add(Button_3, "base")

-- блокируем взаимодействие со всеми кнопками 
GUI.Manager:setProperty("base", "locked", true)

Выполнить общий метод у элементов группы (лучше сделать проврку по типу)

GUI.Manager:callAll(nameGroup, action, ...)

Описание:

  • nameGroup - имя группы, что вы хотите удалить
  • action - какой метод вы хотите вызвать у группы объектов
  • ... - параметры, передоваемые в этот метод
while true do
    local _, side, x, y = os.pullEvent("monitor_touch")
    
    -- проверяем, кликнули ли по кнопке или нет
    GUI.Manager:callAll("base", "onClick", side, x, y)
end

GUI.Label(mon, pos, text, align, ...)

обьект для отображения различной текстовой информации как горизонтально, так и вертикально. имеется поддержка\n

Описание:

  • mon - монитор (или терминал) на котором будет отображатся объект
  • pos - позиция на экране (x, y)
  • text - тект для отображения
  • align - способ отображения текста

      • align[1] - отображение относительно координат pos

    • 1 - влево от pos (при align[2] = 2 вниз)
    • 2 - по серединет pos
    • 3 - вправо от pos (при align[2] = 2 верх)

      • align[2]

    • 1 - горизонтально
    • 2 - вертикально
  • ... - одна и более групп в которые вы хотите добавить element
Метод Описание
:print(text) функция для отображения объекта.

отобразить объект.

GUI.Label:print(text)

Описание:

  • text - задать текст для отображения

Пример: 

-- создаем объект типа Label
local Label = gui.Label(term, { 5, 5 }, "text", { 1, 1 }, "base")

-- term = терминал
-- { 5, 5 } = позиция
-- "text" = ну вы понялм
-- { 1, 1 } = отображение горизонтальное и влево от pos
-- "base" = группа в GUI.Manager (не обязательно)

GUI.Button(mon, pos, text, align, func, ...)

кнопка, вы же знаете что такое кнопка? нет?кнопка область экрана для выполения каких либо функций

Описание:

  • mon - монитор (или терминал) на котором будет отображатся объект
  • pos - позиция кнопки
  • text - - текст кнопки, не обязательна
  • align - тоже что и в Label, но есть нюанс

      • У align есть несколько вариаций

    • align[1] это как у Label
    • align[2] вот тут весело. тут может быть 1, 2 и 4 параметра
    • если align[2], это 1 параметр то это расширение зоны кнопки на 1 симвл
    • если align[2], это 2 параметра, то 1 это расширение в стороны, в 2 это расширение в верх\низ
    • если align[2], это 4 параметра, то это расширения в каждую сторону по отдельности (лева,право,верх,низ)
  • func - функция, выпалняемая при нажатии по кнопке
  • ... - одна и более групп в которые вы хотите добавить element
общие свойства значение описание
locked false отвечает за блакировку взаимодействия с элементов, как графически так и програмно
checked false отвечает показывает взаимодействие с кнопкой
func передаваемя функция функция которая будет выпалнятся при активации обьекта. (когда checked = true)
Метод Описание
:print() функция для отображения объекта.
:onClick(mon, x, y) метод вызова функции при клике по экрану

отобразить объект.

GUI.Button:print()

Пример: 

 -- создаем кнопку 
local Button = gui.Button(term, { 5, 5 }, "text" {1})

-- создаем цикл на проверку нажатия 
while true do
    local _, side, x, y = os.pullEvent("monitor_touch")
    Button:onClick(side,x,y)
end

 -- можно создаем несколько кнопок 
local Button_1 = gui.Button(term, { 5, 5 }, "text" {1},nil,"base")
local Button_2 = gui.Button(term, { 5, 5 }, "text" {1},nil,"base")
local Button_3 = gui.Button(term, { 5, 5 }, "text" {1},nil,"base")

-- создаем цикл на проверку нажатия всех кнопок
while true do
    local _, side, x, y = os.pullEvent("monitor_touch")
    gui.Manager:callAll("base", "onClick", side, x, y)
end

 -- можно заблакировать кнопку, чтобы проверка на нажатия не срабатывала 
Button_1.locked = true
