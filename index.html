GUI Lib для CC: Tweaked

Начало

Данная библиотека создана для создания различных интерфейсов с помощью ui элементов для мода CC:Tweaked. хотите иделать панель управления домом, фермами, реактором из модов в конце концов? это все возможно. хотет сделать информациооное табло? тоже не проблема. написал все один раз и кайфуешь

Версия библиотеки является сырой, поэтому в ней еще могут быть ошибки. Вы можете сообщать обо всех них в канале Discord под ником Lord_Zif_Tech.

Как начать ей пользоватся?

вставте вот эту ссылку в ваш терминал wget https://raw.githubusercontent.com/lord-Vampirion-Tech/cct-gui-lib/main/GUI.lua GUI.lua и библиотека установится сама с репозиторя Git Hub https://github.com/lord-Vampirion-Tech/cct-gui-lib

в своей программе просто пропишите local gui = dofile("GUI.lua") и можете пользоватся всеми благами библиотеки. Все модули, предоставляемые библиотекой, отображаются в боковой панели.

Manager

объект для работы с группой объектов gui библиотеки.

методы:

Добавить элемент в группу(ы).

GUI.Manager:add(element, ...) 
local Label = {...}
GUI.Manager:add(Label, "base")

Удалить всю группу

GUI.Manager:delAll(nameGroup) 
GUI.Manager:delAll("base")

Удалить элементы группы по определенным условиям, которые вы укажите.

GUI.Manager:delIf(nameGroup, predicate) 
local Label = {...} -- обьект типа Label
local Button = {...} -- обьект типа Button

-- добавляем Label в группу base
GUI.Manager:add(Label, "base") 
GUI.Manager:add(Button, "base") 

-- пример удаления по типу
GUI.Manager:delIf("base", function(e)
    return e.type == "Button"
end)

Установить цвет для элементов группы

GUI.Manager:setColors(nameGroup, colors) 
GUI.Manager:setColors("base", {
    back = colors.black,
    text = colors.blue,
})

Изменить своство у элементов группы по определенным условиям

GUI.Manager:setProperty(nameGroup, prop, value) 
-- обьекты типа Button
local Button_1 = {...}
local Button_2 = {...}
local Button_3 = {...}

-- добавляем Button's в группу base
GUI.Manager:add(Button_1, "base") 
GUI.Manager:add(Button_2, "base")
GUI.Manager:add(Button_3, "base")

-- блокируем взаимодействие со всеми кнопками 
GUI.Manager:setProperty("base", "locked", true)

Выполнить общий метод у элементов группы (лучше сделать проврку по типу)

GUI.Manager:callAll(nameGroup, action, ...) 
while true do
    local _, side, x, y = os.pullEvent("monitor_touch")
    
    -- проверяем, кликнули ли по кнопке или нет
    GUI.Manager:callAll("base", "onClick", side, x, y)
end
